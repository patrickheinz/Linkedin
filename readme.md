# 🚀 Códigos dos Posts do LinkedIn

Este repositório é a central de todos os scripts, queries e fluxos de dados que compartilho no meu LinkedIn. 
Meu objetivo é preencher a lacuna entre conceitos teóricos de dados e aplicações reais de negócios. Fique à vontade para explorar, fazer um fork e usar esses trechos de código!
